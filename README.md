# WhoDriver Frontend

Mobile-first интерфейс для WhoDriver — анонимных оценок и отзывов о водителях по российскому госномеру.

## Возможности

- Поиск и нормализация номера: латинские визуальные аналоги автоматически приводятся к кириллице.
- Лента карточек с сортировкой по свежести, рейтингу и числу отзывов; есть пагинация.
- Карточка номера с анонимной оценкой от 0 до 5.
- Добавление, редактирование и удаление своего комментария; жалобы на чужие отзывы.
- Mobile-first адаптивная вёрстка, доступные подписи форм и состояния интерфейса.

## Запуск

Требуется Node.js 22.13+.

```bash
cp .env.example .env
npm install
npm run dev
```

По умолчанию приложение использует API по адресу `http://localhost:3000/api/v1`.
Измените `NEXT_PUBLIC_API_BASE_URL` в `.env`, если бэкенд доступен по другому адресу.

## Интеграция с API

Клиент отправляет запросы с `credentials: "include"`, чтобы браузер передавал анонимную `HttpOnly` cookie WhoDriver.
Для раздельного размещения фронтенда и бэкенда добавьте домен фронтенда в `CORS_ORIGIN` бэкенда и разрешите запросы с credentials.

Если API временно недоступен, интерфейс показывает демонстрационные карточки, чтобы страница оставалась наглядной во время разработки.
